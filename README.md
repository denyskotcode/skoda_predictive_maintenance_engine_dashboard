# 🔧 Škoda Predictive Maintenance System

Pet project for my ML portfolio. Predicting equipment failures on manufacturing lines
using sensor data from the AI4I 2020 dataset.

## Stack
- Python, scikit-learn, XGBoost, LightGBM, SHAP, Streamlit, Plotly

## Status
🚧 Work in progress

## Quick Start
```bash
pip install -r requirements.txt
python train.py
streamlit run streamlit_app.py
```
